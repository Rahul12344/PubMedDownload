a myxozoan parasite belonging genus thelohanellus kudo, 1933 (cnidaria,
myxosporea, bivalvulida) isolated fins labeo dero inhabiting
ranjit sagar wetland, punjab, india. plasmodium 0.5-0.7 mm diameter
each containing 80- 100 number myxospores. myxospores egg shaped to
ovoidal valvular view bluntly pointed anterior broad rounded
posterior end measuring 8.36x 4.77 µm. polar capsule single, broadly pyriform
in shape, measuring 4.77x3.98 µm size containing polar filament coiled
perpendicular longitudinal axis myxospore body making 7-9 turns. blast 
analysis 18s rdna sequence isolate demonstrated 98% homogeneity with
thelohanellus sp. (ku155561, unpublished), followed 91% t. caudatus
(km252684) thelohanellus sp. fco (kr819273). intensity infection was
recorded light indicated fin plasmodial index (fpi = 1). the
present study, t. boggoti described using 18s rrna gene phylogenetic
analysis using mega.6.
