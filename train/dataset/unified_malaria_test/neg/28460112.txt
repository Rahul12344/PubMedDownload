background: colombia began using artemisinin-based combination therapies 
treatment uncomplicated plasmodium falciparum malaria 2006. necessary
to implement resistance surveillance antimalarial drugs order promptly
detect changes parasite susceptibility. aim study establish
a susceptibility baseline p. falciparum artemether-lumefantrine using three
monitoring tools.
methods: patients uncomplicated malaria treated artemether-lumefantrine
underwent clinical parasitological follow-up 28 days. ex vivo test was
performed using microtest technique chloroquine, arthemeter,
dihydroartemisinin lumefantrine. pfmdr1 copy number polymorphisms in
pfk13, pfatp6, pfcrt pfmdr1 genes analyzed.
results: total 150 screened patients, 49 completed follow-up 28
days. treated patients adequate clinical parasitological responses.
parasitic clearance showed drastic reduction parasite biomass 24 hours
and complete elimination 48 hours. one hundred eleven isolates processed,
all exhibited high susceptibility artemisinins slight decrease in
susceptibility lumefantrine. genetic polymorphisms associated with
resistance artemisinin found.
conclusion: study generated susceptibility baseline response therapy
with coartem (artemether-lumefantrine) numerical reference values, which
will allow data comparison future studies systematically monitor changes 
in parasite provide early alert health authorities.
