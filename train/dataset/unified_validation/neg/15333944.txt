triatoma virus (trv) viral pathogen blood-sucking reduviid bug
triatoma infestans, important vector american human trypanosomiasis
(chagas' disease). trv putatively classified member the
cripavirus genus (type cricket paralysis virus) dicistroviridae family.
this work describes purification trv particles infected t. infestans 
and crystallization preliminary crystallographic analyses. two
different crystal forms, rhombohedral orthorhombic, obtained room
temperature hanging-drop vapour-diffusion technique using polyethylene
glycol polyethylene glycol monomethylether precipitants. rhombohedral 
crystals unit-cell parameters = b = 306.6, c = 788.4 (hexagonal
setting), diffract 3.2 resolution contain one-third viral
particle per asymmetric unit. orthorhombic crystals cell parameters = 
336, b = 351, c = 332 a, diffract 2.5 resolution, contain one-half
of virus particle asymmetric unit. complete diffraction data set has
been collected 3.2 resolution, using synchrotron radiation, single
rhombohedral crystal cryogenic conditions.
