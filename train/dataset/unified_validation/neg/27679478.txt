the influenza research database (ird) u.s. national institute allergy and
infectious diseases (niaid)-sponsored bioinformatics resource center dedicated to
providing bioinformatics support influenza virus research. ird facilitates
the research development vaccines, diagnostics therapeutics against
influenza virus providing comprehensive collection influenza-related data
integrated various sources, growing suite analysis visualization
tools data mining hypothesis generation, personal workbench spaces for
data storage sharing, active user community support. here, describe
the recent improvements ird including use cloud high performance
computing resources, analysis visualization user-provided sequence data
with associated metadata, predictions novel variant proteins, annotations of
phenotype-associated sequence markers predicted phenotypic effects,
hemagglutinin (ha) clade classifications, automated tool ha subtype
numbering conversion, linkouts disease event data addition host
factor antiviral drug components. data tools freely available
without restriction ird website https://www.fludb.org.
