background: plasmodium vivax duffy binding protein (pvdbp) 
studied ligand binding human reticulocytes date. molecule a
cysteine-rich domain region ii (rii) used control for
evaluating target cell binding activity several parasite molecules.
however, obtaining rpvdbp-rii soluble form using escherichia coli
expression system usually requires laborious time-consuming steps for
recovering molecule's structure function, considering extracted
from inclusion bodies. present study describes easy fast method for
expressing obtaining several pvdbp fragments prove ideal use
in protein-cell interaction assays.
results: two pvdbp encoding regions (rii riii/v) cloned pexp5-ct
vector expressed e. coli extracted soluble fraction
(rpvdbp-riis rpvdbp-riii/vs) using simple freezing/thawing protocol. 
the purification, dichroism analysis enabled verifying high rpvdbp-riis and
rpvdbp-riii/vs secondary structure α-helix content, lowered when
molecules extracted inclusion bodies (rpvdbp-riiib rpvdbp-riii/vib)
using denaturing step. interestingly, rpvdbp-riis, rpvdbp-riiib, bound 
to human reticulocytes, rpvdbp-riii/vs rpvdbp-riii/vib bound such
cells similar way negative control (cells incubated without recombinant
proteins).
conclusions: research shown first time rpvdbp-rii be
expressed obtained soluble form using e. coli system avoiding 
denaturation refolding steps commonly used. results highlight the
usefulness rpvdbp-riii/vs fragment non-binding control for
protein-cell target interaction assays. soluble extraction protocol described
is good alternative obtain fully functional p. vivax proteins fast 
easy way, surely prove useful laboratories working studying this
parasite's biology.
